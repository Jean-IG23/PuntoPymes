# Diagramas de Casos de Uso (Mermaid)

Estos archivos contienen diagramas de casos de uso por módulo en formato Mermaid. Puedes visualizarlos en cualquier editor o plataforma que soporte Mermaid (por ejemplo: VS Code con la extensión "Markdown Preview Mermaid Support", o en GitHub si se renderiza). 

Archivos:

- `docs/diagrams/01-autenticacion.md`
- `docs/diagrams/02-empleados.md`
- `docs/diagrams/03-asistencia.md`
- `docs/diagrams/04-tareas.md`
- `docs/diagrams/05-objetivos.md`
- `docs/diagrams/06-nomina.md`

Cómo previsualizar rápido (en VS Code):

1. Instala la extensión "Markdown Preview Mermaid Support".
2. Abre cualquiera de los archivos `.md`.
3. Pulsa `Ctrl+Shift+V` para previsualizar.

Si quieres que los guarde como `.mmd` separados en lugar de `.md`, dímelo y los actualizo.
